# Global Macro Signal - 要件定義書 (Pro Version)

5人の専門家チームによる再定義を行い、機関投資家レベルの「プロ仕様」ダッシュボードを目指します。

## 1. 金融クオンツ（指標・ロジック）- Advanced
**目標**: 市場の「歪み」と「危機」を早期検知するマクロリスクモデルの構築。

*   **リスク信号 (Risk ON/OFF Signal)**:
    *   **HY OAS (ハイ・イールド債スプレッド)**: 信用収縮の早期警戒信号 (FRED)
    *   **VIX (恐怖指数)**: 株式市場の変動リスク
    *   **MOVE指数**: 債券市場の恐怖指数（VIXの債券版）
*   **先行指標 (Leading Indicators)**:
    *   **逆イールド (Yield Curve Inversion)**:
        *   10年債 - 2年債 (10Y-2Y)
        *   10年債 - 3ヶ月債 (10Y-3M)
        *   ※過去の発生履歴データベースを構築し、リセッション確率を算出
*   **市場内部構造 (Market Structure)**:
    *   **ヒンデンブルグ・オーメン**: 暴落の前兆シグナル
    *   **米ネット流動性 (Net Liquidity)**: Fed Balance Sheet - TGA - RRP (株価との連動性が高い)
    *   **銅/金レシオ (Copper/Gold Ratio)**: 景気先行指標

## 2. SRE（信頼性・運用）- Robust
**目標**: 複数のデータソースを統合し、欠損なきデータパイプラインを構築する。

*   **ハイブリッドデータソース**:
    *   **FRED (St. Louis Fed) API**: マクロ経済データ (HY OAS, 金利, ネット流動性要素) の取得
    *   **Yahoo Finance (yfinance)**: 株価、VIX、先物データの取得
*   **冗長性と整合性**:
    *   各ソースの更新タイミングのズレを吸収するデータ正規化処理
    *   APIレート制限（アクセス制限）を回避するスマートな再試行ロジック

## 3. UI/UXデザイナー（世界基準の直感性）- Cockpit
**目標**: 複雑なマクロ指標を「一瞬」で判断させる視覚言語。

*   **シグナルパネル (Signal Panel)**:
    *   トップ画面最上部に **[ 🟢 Safe / 🟡 Caution / 🔴 Danger ]** の総合判定を表示
    *   各サブ指標（HY OAS, VIX等）ごとの現在ステータスをアイコン表示
*   **高度なチャート表現 (Plotly)**:
    *   **オーバーレイチャート**: 「S&P500」と「ネット流動性」を重ねて乖離を表示
    *   **インタラクティブ機能**: ズーム、期間指定、特定ポイントのツールチップ表示

## 4. グロース・SEO・AI戦略 - Global & Automated
**目標**: 4ヶ国語展開 × AI自動生成記事で、世界の検索流入を独占する。

*   **多言語対応**:
    *   **日本語、英語(English)、中国語(中文)、スペイン語(Español)**
*   **AIコンテンツファクトリー**:
    *   **Gemini 3 Pro** を活用し、毎日データ更新直後に「本日の市場分析レポート」を各言語で自動執筆
    *   単なる事実の羅列ではなく、「HY OASが急上昇しており、警戒が必要」といったプロレベルの文脈解釈を含める
*   **SEO構造**:
    *   各指標（例: `/en/indicators/yield-curve`）ごとの解説ページを自動生成し、ロングテールキーワードを狙う

## 5. リーガルアドバイザー（法的リスクの盾）
**目標**: グローバル展開に耐えうる法的堅牢性の確保。

*   **国際的な免責事項**:
    *   各言語に応じた適切な免責文言の表示
*   **AI生成コンテンツの明示**:
    *   「解説記事はAIによって生成されたものであり、完全性を保証しない」旨のディスクレーマー

### 技術選定の変更点（VBAユーザー向け解説）
前回の計画から、さらに高度なツールを追加します。

1.  **FRED API**: 米国政府（連銀）が提供する公式データの接続口。VLOOKUPの参照先が「Yahoo個人のページ」から「財務省の公式帳簿」に変わるイメージです。
2.  **Plotly (プロットリー)**: Excelのグラフ機能の上位互換。Web上でグリグリ動かせるリッチなグラフを作ります。
3.  **Gemini 3 Pro API**: あなたの代わりに毎日レポートを書いてくれる「超優秀な専属アナリスト」をプログラムに組み込みます。
